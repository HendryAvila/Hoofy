# Hoofy Plugin for Claude Code

> **This directory is a historical reference only.**
> The source of truth for the Hoofy Claude Code plugin is now the marketplace repo:
>
> **[HendryAvila/hoofy-plugins](https://github.com/HendryAvila/hoofy-plugins)**

## Installation

Inside Claude Code, run:

```
/plugin marketplace add HendryAvila/hoofy-plugins
/plugin install hoofy@hoofy-plugins
```

See the [marketplace repo](https://github.com/HendryAvila/hoofy-plugins) for full documentation.
